# Security Policy

## Supported versions

Security fixes are handled on the default branch and included in the next release.
Use the latest released version of Helixiora Endpoint Security unless Helixiora has
provided a different build internally.

## Reporting a vulnerability

Report vulnerabilities through GitHub private vulnerability reporting for this
repository. Do not open a public issue for suspected vulnerabilities.

Include:

- Affected platform or workflow
- Steps to reproduce
- Expected and observed behavior
- Impact and any known exploit prerequisites

Helixiora will triage reports privately and coordinate fixes before public
disclosure where needed.
